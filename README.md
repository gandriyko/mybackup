mybackup
============

Script for backing MySQL (MariaDB) databases.

## OPTIONS
    -h hostname
    -p interactive password
    -P password
    -u username
    -v verbose
    -d database 
    -l limit rows


wrapper for mybackup (save)
=================================================================

Script for coping backup to remote server via SSH

## OPTIONS
    -h hostname
    -p interactive password
    -P password
    -u username
    -v verbose
    -d database
    -l limit rows
    -r copy to remote server
