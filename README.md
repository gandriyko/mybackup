# mybackup

Script for backing MySQL (MariaDB) databases.

use mybackup:
-h hostname
-p interactive password
-P password
-u username
-v verbose
-d database 
-l limit rows


wrapper for mybackup (for coping backup to remote server via SSH)
save:
-h hostname
-p interactive password
-P password
-u username
-v verbose
-d database
-l limit rows
-r copy to remote server
